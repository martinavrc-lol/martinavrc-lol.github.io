# Martina's Portfolio

A retro, terminal-styled personal portfolio built from scratch with vanilla
HTML, CSS & JavaScript — no frameworks, no libraries.

🔗 **Live:** https://martinavrc-lol.github.io/

## About

A playful portfolio with a warm, retro aesthetic (dashed borders, a burgundy &
peach palette and terminal-style prompts). It showcases my web design and
front-end work, including three full WordPress case studies.

## Features

- Fully responsive (desktop, tablet & mobile)
- Hamburger navigation menu
- Animated floating stars in the hero (requestAnimationFrame)
- Scroll-reveal effects (IntersectionObserver)
- Image "deck" carousel on each project page
- Back-to-top floating button
- Custom star cursor
- Optimized images (WebP + JPG via `<picture>`)
- Self-hosted + Google Fonts, AA-contrast colors

## Built with

- **HTML5** — semantic markup
- **CSS3** — Flexbox, Grid, custom properties, media queries
- **JavaScript (ES6)** — vanilla, no dependencies
